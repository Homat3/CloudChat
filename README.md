# 项目说明

1. **项目简介**
    - CloudChat 云聊
    - 包含子项目：
        - Web 客户端子项目 (Angular)
        - HarmonyOS 6 客户端子项目 (ArkUI)
        - Ubuntu 服务端子项目 (C++)

2. **项目仓库**
    - Web 客户端子项目([Github: CloudChat](https://github.com/Homat3/CloudChat))
    - HarmonyOS 6 客户端子项目([Omniedu-GitLab: D2xn61EZeEB82](https://gitlab.omniedu.com/root/D2xn61EZeEB82.git))
    - Ubuntu 服务端子项目([Omniedu-GitLab: D2xn61EZeEB82](https://gitlab.omniedu.com/root/D2xn61EZeEB82.git))
