# 项目说明

1. **项目简介**
    - CloudChat 云聊
    - 包含子项目：
        - Web 客户端子项目 (Angular)
        - HarmonyOS 6 客户端子项目 (ArkUI)
        - Ubuntu 服务端子项目 (C++)

2. **项目仓库**
    - [Web 客户端子项目](./src/web/)
    - [HarmonyOS 6 客户端子项目](./src/client/)
    - [Ubuntu 服务端子项目](./src/server/)

# 本仓库结构

```
CloudChat/
├─ README.md                # 说明文档（本文档）
├─ docs                     # 约定文档
└─ src/
   ├─ client/               # HarmonyOS 6 客户端子项目
   ├─ web/                  # Web 客户端子项目
   └─ server/               # Ubuntu 服务端子项目
```
