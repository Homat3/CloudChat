# 色彩资源使用规范
## 前言
本文为 CloudChat 鸿蒙版开发 资源使用规范文件-色彩篇

本文将规定开发过程中，对色彩资源的使用方法

## 使用范围
在各个标记了 @Component 注解的组件对象中使用到的 UI 色彩

即需要呈现给客户的色彩的大部分颜色

少数颜色需要用固定颜色，可以使用 Color.XXXX 来调用例如：
```extendtypescript
TextInput().backgroundColor(Color.White)
```


## 使用方法

1. 找到色彩资源文件定义文件： [默认主题 color.json](entry/src/main/resources/base/element/color.json) 和 [深色主题 color.json](entry/src/main/resources/dark/element/color.json)
2. 在其中根对象的"color"列表中按照如下格式添加新的色彩资源：
    ```json5
    {
      "name": "{page-name}.{component-type}.{detailed-property}.{detailed-property}...",
      "value": "XXXXXX"
    }
    ```
   其中

   page-name 表示用到这个色彩资源的页面名称（如 login，contact-list），采用英文字母小写+连字符(-)的形式

   component-type 表示这个色彩资源所在的组件类型（如 text(文本), textInput(文本输入框), button(按钮)），用小驼峰命名，和调用的组件函数一致

   detailed-property 表示这个色彩资源描述的组件属性名可以一直叠加（如 border(边框), font(字体)），用小驼峰命名，和调用的组件属性函数一致
3. 接下来在代码中调用，调用分两种情况，见[在使用 build() 函数直接描述 UI 的情况下](#1-在使用-build-函数直接描述-ui-的情况下)和[在其它方法的处理过程中用到的情况下](#2-在其它方法的处理过程中用到的情况下)

### (1) 在使用 build() 函数直接描述 UI 的情况下
1. 首先在色彩资源文件中定义好相关资源(见[使用方法](#使用方法))
2. 直接使用 $r() 函数来引用资源

   例如：

   现在有如下色彩资源定义分别位于[默认主题 color.json](entry/src/main/resources/base/element/color.json) 和 [深色主题 color.json](entry/src/main/resources/dark/element/color.json)
    ```json5
    {
      "name": "login_textInput_all_border",
      "value": "#DCDFE6"
    }
    ```
    ```json5
    {
      "name": "login_textInput_all_border",
      "value": "#606266"
    }
    ```
   你在 build() 函数中应该如此使用它，例如一个文本输入框控件的背景色：
    ```extendtypescript
    TextInput().backgroundColor($r(login_textInput_all_border))
    ```

### (2) 在其它方法的处理过程中用到的情况下
1. 首先在色彩资源文件中定义好相关资源(见[使用方法](#使用方法))
2. 直接使用 [util组件](entry/src/main/ets/core/util) 中定义的[load-resource系列函数](entry/src/main/ets/core/service/resource/load-resource.ets)来引用资源

   例如：

   现在有如下色彩资源定义分别位于[默认主题 color.json](entry/src/main/resources/base/element/color.json) 和 [深色主题 color.json](entry/src/main/resources/dark/element/color.json)
    ```json5
    {
      "name": "login_textInput_all_border",
      "value": "#DCDFE6"
    }
    ```
    ```json5
    {
      "name": "login_textInput_all_border",
      "value": "#606266"
    }
    ```
   你在 private example() 函数中应该如此使用它，例如加载并输出色彩资源文本：
    ```extendtypescript
    struct example{
      private example(): string{
        const color = loadColorResource(this.uiAbilityContext, $r(app.login_textInput_all_border));
        return color;
      }
    }
    ```
