# 客户端->服务端 消息类型

| 消息类型     | 包含信息                                   |   |
|----------|----------------------------------------|---|
| 请求加载消息   | 请求者的用户 id                              |   |
| 请求发送普通消息 | 请求者的用户 id<br/>消息内容                     |   |
| 请求发送文件消息 | 请求者的用户 id<br/>文件名称<br/>文件内容(二进制组)<br/> |   |
| 请求发送图片消息 | 请求者的用户 id<br/>图片名称<br/>图片内容(二进制组)<br/> |   |
| 请求标记消息已读 | 请求者的用户 id                              |   |
| 请求清除历史消息 | 请求者的用户 id                              |   |

# 服务端->客户端 消息类型


| 消息类型       | 包含信息 |   |
|------------|------|---|
| 消息加载完成     | 无    |   |
| 回应已经收到消息   | 无    |   |
| 回应已经清除历史消息 | 无    |   |
